HTML Paragraphs

The HTML < p > element defines a paragraph.

A paragraph always starts on a new line, and browsers automatically add some white space before and after a paragraph. Also remove any extra spaces and lines when the page is displayed.

Horizontal Rules

The < hr > tag defines a thematic break in an HTML page, and is most often displayed as a horizontal rule.

< hr > is used before after this section as a result a horizontal line is displayed before and after this section.

Line Breaks

Use < br > if you want a line break (a new line) without starting a new paragraph. The < br > tag is an empty tag.

Example:
This is
a paragraph
with line breaks.

Preformatted Text

The text inside a < pre > element is displayed in a fixed-width font (usually Courier), and it preserves both spaces and line breaks.

Example 
 This    is       a        paragraph.
