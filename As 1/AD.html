MARQUEE

The < marquee > tag is used to create a scrolling effect for text or images. This tag can make content move left, right, up, or down, adding an interactive element to your web page.

Syntax: 
	< marquee >Scrolling text goes here!< /marquee >
Marquee tag is not only applied on text but can also be applied on image and links.

However, the < marquee > tag is now considered obsolete and non-standard. It is not recommended for use in modern web development because it does not align with current HTML standards and can lead to accessibility and usability issues.


Marquee on image

Syntax: 
	< marquee > < img src = "img.jpg" > < /marquee >



Marquee on link

Syntax: 
	< marquee >< a href="link.html" >This is a link< a >< /marquee >
