HTML Images

Images can improve the design and the appearance of a web page.

The < img > tag is a empty tag, it contains attributes only, and does not have a closing tag.

The tag has two required attributes:

src - Specifies the path to the image.
alt - Specifies an alternate text for the image.

Syntax:
        < img src="url" alt="alternate text" >

Image Property

=> Image Size
-The width and height attributes always define the width and height of the image in pixels.

< img src="html.png" alt="HTML 5" width="200" height="200" >

=> Image as a link
-To use an image as a link, put the < img > tag inside the tag:

< a href="example.html" >
< img src="html.png" alt="HTML 5" >
< /a>
