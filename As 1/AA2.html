HTML HEADINGS

HTML headings are titles or subtitles that you want to display on a webpage.

HTML headings are defined with the < h1 > to < h6 > tags.

< h1 > defines the most important heading. < h6 > defines the least important heading.

TAG EFFECT
< h1 >

This is the first type of heading.
< h2 >

This is the second type of heading.
< h3 >

This is the third type of heading.
< h1 >

This is the fourth type of heading.
< h5 >
This is the fifth type of heading.
< h6 >
This is the sixth type of heading.
