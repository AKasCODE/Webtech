1. Bold Tag

< b > tag is used to bold the text.

"This text is bold."

2. Italic Tag

< i > tag is used to display the text in italic. Also defines a part of text in an alternate voice or mood.

"This text is italic."

3. Underline Tag

< u > tag is used to underline the text.

"This is a HTML document."

4. Superscript

< sup > tag displays the text half a character above the normal line.

"This is superscripted text."

5. Subscript

< sub > tag displays the text half a character below the normal line.

"This is subscripted text."

6. Strong Tag

< strong > tag defines the strong importance a text. The content is displayed in bold.

"This text is important!"

7. Emphasized Tag

< em > is used to display emphasized text. The content is displayed in italic.

"This text is emphasized." Tip: A screen reader will pronounce the words in < em > with an emphasis, using verbal stress.

8. Small Tag

< small > tag is used to display the smaller text.

"This is some smaller text."

9. Delete Tag

< del > tag strikes a line through the text. It defines that the text has been deleted from a document.

"This is a CSS HTML document."

10. Mark Tag

< mark > tag is uesd to highlight the text.

"Do not forget to buy milk today."
